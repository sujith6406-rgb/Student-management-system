# Student Management System

A web-based Student Management System built using Flask and SQLite.

## Features
- Add Students
- Edit Student Details
- Delete Students
- Search Students
- Dashboard with Statistics

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/sujith6406-rgb/student-management-system.git
cd student-management-system
pip install -r requirements.txt
python app.py
```

Visit:
http://127.0.0.1:5000
